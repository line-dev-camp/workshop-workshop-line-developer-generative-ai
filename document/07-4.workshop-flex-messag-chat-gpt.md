### แนวทาง

1. ใช้ ChatGPT เพื่อออกแบบและเขียนโค้ด JSON สำหรับ Flex Message
2. ทดสอบ Flex Message ใน LINE Developer Console หรือใน LINE Bot ของคุณ
3. ตรวจสอบว่าข้อมูลทุกส่วนถูกต้องและสามารถแสดงผลได้อย่างถูกต้อง
